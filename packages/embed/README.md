# @strapi-forms/embed

Lightweight (< 20KB gzipped) embed snippet for the Strapi Forms plugin. Drop one `<script>` tag onto any page and a working form renders, validates, and submits.

```html
<div data-strapi-form="contact" data-strapi-base-url="https://cms.example.com"></div>
<script src="https://cdn.jsdelivr.net/npm/@strapi-forms/embed/dist/embed.iife.js"></script>
```

See `docs/embed-snippet.md` in the monorepo root for full usage and the CSS hook contract.
