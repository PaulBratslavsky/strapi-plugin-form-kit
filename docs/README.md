# Documentation

extending forms in Strapi v5.

New here? Start with **[Getting started](getting-started.md)** — install,
build your first form, embed it.

- **[Getting started](getting-started.md)** — install, configure, build,
  publish, embed in five minutes.
- **[AI form builder](ai-builder.md)** — draft and refine forms from

  (Anthropic / OpenAI / Ollama / Mock).
- **[The canonical form schema](form-schema.md)** — the JSON contract
